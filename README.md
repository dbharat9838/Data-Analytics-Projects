📁 Projects

1️⃣ Employee Performance & Retention Analysis

Tools Used: Python, Power BI

Dataset: ~1,470 employee records with 35+ attributes

Objective

Analyze employee data to identify retention patterns and performance trends to support HR decision-making.

Key Insights

- Identified departments with the highest attrition rate (~16%)
- Employees with low job satisfaction had nearly 2× higher churn probability
- Developed an interactive HR dashboard with 10+ KPIs, including attrition rate, department performance, and employee demographics

2️⃣ Financial Performance Analysis

Tools Used: Python, Power BI

Dataset: Financial records including revenue, expenses, and profit metrics

Objective

Analyze financial performance to understand profitability trends and cost distribution.

Key Insights

- Analyzed 12 months of financial data across multiple product categories
- Identified top-performing revenue streams contributing ~35% of total revenue
- Created a Power BI dashboard with 8+ KPIs, including revenue trend, profit margin, and cost breakdown

3️⃣ Fraud Transaction Analysis

Tools Used: Python (Pandas, Matplotlib, Seaborn)

Dataset: Transaction dataset containing 280,000+ records

Objective

Identify patterns in fraudulent transactions using exploratory data analysis techniques.

Key Insights

- Fraud transactions represented less than 1% of total transactions
- Fraud cases were concentrated in specific transaction types and time periods
- Identified behavioral patterns useful for fraud detection models
